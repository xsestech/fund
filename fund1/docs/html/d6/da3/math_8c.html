◆ factorial()

long long int factorial ( const int number)

Nothing more than simple factorial func.

Actually, I found out, than researching for faster algo, that factorial is not O(n), because of the multiplication. As \( n \rightarrow \inf \), the complexity of multiplication starts to grow to \( O(n^2) \), because on large number hardware acceleration stops to work. But it can be fixed with Schönhage–Strassen algorithm, giving \( O(nlogn) \) complexity
