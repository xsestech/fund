Detailed Description

Module that checks format of cli arguments and call corresponding funcs.

This module was built for one argument cli with one or more params. Checks the correctness on tokens level and calls function from config. You can pass args with '-' or '/' symbol. For example, {

./program -h 10 10} is
correct, {@code./program /h 10 10

} is also corrent, but {

./program -h 10 10 -p 1

} is incorrect
