## Docs
Code documentation is available on https://xsestech.github.io/fund/
## Usage
### Prerequisites
You will need:
- llvm clang with google santizers
- cmake 3.25 installed
### Building
```bash
cmake -S . -B build
cmake --build build
```
## Running
```bash
./build/taskN/taskN
```
